# Inviter teammedlemmer

*Tidsbruk: ca. 1 minutt*

## Oversikt

Inviter kollegaer til organisasjonen slik at de kan se og jobbe med samme klienter.

## Steg 1: Åpne innstillinger

1. Klikk på organisasjonsnavnet eller **Innstillinger** i sidemenyen.
2. Gå til **Medlemmer** eller **Team**.

## Steg 2: Send invitasjon

1. Klikk **Inviter medlem**.
2. Fyll inn e-postadressen og velg rolle (admin eller medlem).
3. Invitasjonen sendes på e-post; mottakeren må akseptere for å bli med.

## Roller

- **Admin** — Kan administrere medlemmer og innstillinger.
- **Medlem** — Kan bruke klienter og avstemming i henhold til tilganger.
