# Navigere dashbordet

Kort oversikt over hovedområdene i Revizo.

## Sidemeny

- **Dashboard** — Oversikt og oppgaver
- **Klienter** — Liste over klienter og kontoer
- **Avstemming** — Per klient: mengde 1 og 2, matching, notater
- **Innstillinger** — Organisasjon, Tripletex, team

## Klient-sider

For hver klient finner du:

- **Import** — Last opp filer (Excel, CSV, bank)
- **Avstemming** — Transaksjonspaneler, Smart Match, rapporter
- **Eksport** — PDF- og Excel-rapporter
