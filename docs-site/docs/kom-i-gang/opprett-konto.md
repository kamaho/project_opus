# Opprett konto

*Tidsbruk: ca. 2 minutter*

## Steg 1: Gå til registrering

1. Åpne [app.revizo.no](https://app.revizo.no) og klikk **Registrer deg**.
2. Fyll inn e-post og passord, eller bruk Google / Microsoft for innlogging.

## Steg 2: Bekreft e-post

Sjekk innboksen og klikk lenken for å bekrefte e-posten.

## Steg 3: Fullfør onboarding

Følg stegene i onboarding for å opprette organisasjonen og invitere teammedlemmer.
