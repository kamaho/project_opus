# Feilkoder

| HTTP | Beskrivelse |
|------|-------------|
| 400 | Ugyldig forespørsel (validering) |
| 401 | Manglende eller ugyldig autentisering |
| 403 | Ingen tilgang til ressursen |
| 404 | Ressurs ikke funnet |
| 429 | For mange forespørsler (rate limit) |
| 500 | Serverfeil |

*Utvides med konkrete feilkoder ved API-lansering.*
