# FAQ — Sikkerhet

*Korte svar om sikkerhet og personvern. Detaljert policy kan lenkes her.*

## Hvor lagres data?

I EU med kryptering (Supabase/PostgreSQL).

## Hvem har tilgang?

Kun brukere i din organisasjon som du inviterer. Admin kan styre roller.

## Er dataene kryptert?

Ja, under transport (TLS) og i ro (database og fil-lagring).
