# FAQ — Generelt

*Her samles vanlige spørsmål. Oppdateres ut fra supporthenvendelser.*

## Hvor lagres dataene?

Data lagres sikkert i EU med kryptering. Se [Sikkerhet](/docs/faq/sikkerhet).

## Kan jeg eksportere alt?

Ja. Du kan eksportere rapporter og data i henhold til funksjonene under Eksport for hver klient.
